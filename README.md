## Installation

```bash
$ git clone https://github.com/matheus-pereira/stovepipezera.git
```

```bash
$ cd stovepipezera
```

```bash
$ npm install
```
## Running the app

```bash
$ ng serve
```

Navigate to `http://localhost:4200/`
